# Roadmap

## Sprint 000: Project Brain Bootstrap

Create the documentation structure that future Builders use to understand project state, product intent, architecture, module boundaries, requirements, and active sprint scope.

## Future: Collector Profile Manager

Define and implement the core profile management domain for lifecycle state, profile properties, provisioning, session ingestion, and checkout eligibility.

## Future: Collector Runtime

Define and implement runtime behavior that checks out eligible profiles and performs collection work through adapters without placing automation details in the profile management domain.

## Future: Profile Manager Web UI

Define and implement a management interface for profile lifecycle, provisioning, status inspection, and operational visibility.

## Future: Content Builder

Define and implement the pipeline stage that converts collected material into video outputs.

## Future: Content Publisher

Define and implement the pipeline stage that publishes completed videos to target destinations.
